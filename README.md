# EduLens Student Performance Analytics

## Overview

EduLens Student Performance Analytics is an interactive educational analytics project built using Excel, SQL, Power BI, Python, and Streamlit.

The dashboard helps analyze student academic performance, subject-wise scores, attendance patterns, overall performance trends, and educational insights.

---

## Features

* Student Performance Analysis
* Subject-wise Marks Analysis
* Attendance Analysis
* Academic Trend Analysis
* Performance Insights
* Interactive Dashboard
* KPI Monitoring

---

## Dashboard Metrics

* Average Marks
* Top Score
* Total Students
* Average Attendance Percentage

---

## Visualizations

### Average Marks by Student

Analyze and compare average marks achieved by individual students.

### Subject Performance

Compare student performance across Math, Science, and English subjects.

### Attendance Distribution

Understand attendance categories such as Excellent, Good, and Average.

### Student Distribution

Visualize the overall distribution of students within the dataset.

## Business Insights

* The average student score remained above 80 marks.
* The highest-performing student achieved a score above 95.
* Most students maintained excellent attendance records.
* Student performance remained consistent across all three subjects.
* Attendance showed a positive relationship with academic performance.
